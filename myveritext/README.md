# MyVeritext UI Automation

Hello!

Welcome aboard!
Here you can find automated regression scenarios for MyVeritext, let's check it out!

## Run Locally

Clone the project

```bash
  git clone https://github.veritextcorp.com/Veritext/test-automation-modules
```

Go to the project directory "test-automation-modules"

For MyVeritext, go to its folder
```bash
  cd myveritext
```

Install dependencies

```bash
  npm install
```

If asked, install playwright

```bash
  npx playwright install
```
## Running Tests

To run tests, run the following command

Run on QA
```bash
  cd  tests
  npx playwright test
```

Also, you can verify the documentation below to run with parameters:
https://playwright.dev/docs/running-tests
