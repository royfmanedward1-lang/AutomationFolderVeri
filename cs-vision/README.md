# Commom Services - GDS Automation

Hello!

Welcome aboard!
Here you can find GDS queries automated for Common Services, let's check it out!

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory "test-automation-modules"

For Common Services, go to its folder
```bash
  cd common-services
```

Install dependencies and dotenv

```bash
  npm install
  npm install dotenv
```

If asked, install playwright

```bash
  npx playwright install
```
## Running Tests

To run tests, run the following command

Run on QA
```bash
  cd api-tests
  npx playwright test
```
